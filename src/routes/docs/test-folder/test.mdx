# Installing Wino Mail
## System Requirements

- Windows 10 version 1809 or higher
- 4GB RAM (minimum)
- 100MB free disk space

## Installation Methods

### Microsoft Store (Recommended)

The recommended way to install Wino Mail is through the Microsoft Store:

<a href="https://apps.microsoft.com/detail/9NCRCVJC50WL?mode=full">
	<img src="https://get.microsoft.com/images/en-us%20dark.svg" width="200"/>
</a>

### Preview Releases

To try the latest preview features:

1. Go to [GitHub Releases](https://github.com/wino-mail/wino-mail/releases)
2. Download the latest `.msi` package
3. Run the installer
4. Follow installation wizard steps

### Building from Source

For developers who want to build from source:

1. Clone the repository from GitHub
2. Install development dependencies
3. Build using development tools
4. Run the application

> ⚠️ **Note**: Sideloading is not currently supported. Please use official distribution channels only.
